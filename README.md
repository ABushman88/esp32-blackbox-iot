# ESP32 Telemetry Data Logging Platform

## Overview
This project is a cloud-connected modular black box system that collects sensor data using ESP32 and stores it locally and remotely using a Django-based web application.

---

## Objectives
- Build an embedded system for real-time data logging
- Develop a full-stack web application using Django
- Implement CRUD operations for sensor data
- Support modular sensor expansion
- Apply Agile development methodology

---

## System Architecture
ESP32 → HTTP → Django Backend → Database → Web Dashboard

---

## Technologies
- ESP32 (C++)
- Django (Python)
- SQLite / PostgreSQL
- HTML, CSS, JavaScript

---

## Features
- User authentication (login/register)
- Sensor data logging
- CRUD operations
- Admin dashboard
- Scalable sensor support
